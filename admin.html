İstifadəçi İdarəetmə

Necə işləyir:

  1. Supabase Dashboard → Authentication → Users → "Add User" ilə yeni istifadəçi yaradın
  2. Buradan həmin istifadəçinin User ID-sini müvafiq məkana bağlayın
  3. Müştəri login.html səhifəsindən giriş edəndə avtomatik olaraq öz məkanının panelinə yönləndirilir
